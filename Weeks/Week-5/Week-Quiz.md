# Week 5 - Weekly Knowledge Check

> **Quiz**
> **Q:** When preparing for instance policies and data handling procedures for a ServiceNow rollout, which of the following activities is primarily the responsibility of a data controller?
> **Options:**
> - Managing security contact details and data classification and retention
> - Maintaining physical security controls at colocation facilities
> - Managing media disposal and destruction
> - Issuing TLS certificates for data-center network devices
> **Correct:** Managing security contact details and data classification and retention

> **Quiz**
> **Q:** What is the primary purpose of threat modeling in the design of security systems?
> **Options:**
> - To monitor system activity and detect security breaches in real time
> - To certify that all security tests have been completed successfully
> - To define compliance policies for data retention and access
> - To provide a systematic method for identifying potential security gaps and planning mitigations
> **Correct:** To provide a systematic method for identifying potential security gaps and planning mitigations

> **Quiz**
> **Q:** Which layer of platform security can customers access only through the internet services layer and not directly from the internet?
> **Options:**
> - Network layer
> - Database layer
> - Application layer
> - Physical layer
> **Correct:** Application layer

> **Quiz**
> **Q:** As a CTA, you are advising an organization on access controls for a ServiceNow instance used across multiple business units. To ensure that only authorized users can access sensitive workflows and data, regardless of whether they are on the corporate network or working remotely, which solution should you recommend to enforce least-privilege access while continuously verifying users and devices?
> **Options:**
> - Grant access based on user roles and network location
> - Apply device posture checks for remote users only
> - Configure user access using Zero Trust Access
> - Use static role-based access controls without continuous verification
> **Correct:** Configure user access using Zero Trust Access

> **Quiz**
> **Q:** In ServiceNow, Cloud Encryption is primarily used to protect data in which situation?
> **Options:**
> - When securing database content and backups stored in the datacenter
> - When encrypting data sent between the browser and the instance
> - When masking sensitive information within a specific application field
> - When securing API traffic between ServiceNow and external systems
> **Correct:** When securing database content and backups stored in the datacenter

> **Quiz**
> **Q:** You are a CTA working with a financial services customer that stores private and confidential data such as credit card numbers, government IDs, and contact details in their ServiceNow instance. The customer wants to minimize compliance risk and prevent unauthorized access. Which actions should you recommend to identify and protect this data? (Select all that apply)
> **Options:**
> - Configure scheduled jobs to monitor system performance and notify admins of irregular activity
> - Enable Data Discovery to locate sensitive data stored across instance tables
> - Use Data Privacy to classify and anonymize sensitive data so it is no longer considered regulated private information
> - Use Performance Analytics to monitor activities involving private data over time
> **Correct:** Enable Data Discovery to locate sensitive data stored across instance tables, Use Data Privacy to classify and anonymize sensitive data so it is no longer considered regulated private information
